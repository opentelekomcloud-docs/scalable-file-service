How Do I Access a File System from a Client Server?
===================================================

To access a file system from a Linux server, install the NFS client on the server and run **mount** to mount the file system. To access a file system from a Windows server, install the NFS client on the server, modify the NFS transfer protocol, and run command to mount the file system. Then, you can share the files and directories in the file system.
